# Joystick
InputLibを応用したJoystickライブラリ
**このリポジトリは古いリポジトリです**
**JoystickライブラリはGamePadLibプロジェクトへ統合されました**

## 概要
拙作InputLibを応用したジョイスティック（ゲームパッド）ライブラリです。
シンプルな自作ジョイスティック（ゲームパッド）デバイスを手軽にESP32シリーズで使用することができます。
アナログレバー（X軸/Y軸）、アナログレバー内蔵プッシュスイッチ（Pボタン）、他3ボタン（A,B,Cボタン）をサポートします。

## 特徴
- アナログレバーを十字ボタンのように扱うことができます。もちろんADC値を取得することもできます。
- レバー／ボタンの押下状態を同期／非同期に監視することができます。
- レバー／ボタンの同時押しを検知することができます。
- レバー／ボタンの押下状態に応じたコールバックを登録し、非同期でキー操作に対応することができます。

## フォルダ構成
```
<lib>
  +--<InputLib>
  +--<Joystick>
```
## platformio.ini
下記を追加
```
lib_extra_dirs =
  ./lib/InputLib
  ./lib/Joystick
```

## 動作確認環境
- PlatformIO + ArduinoFramework
- ESP32-C3
- アルプスアルパイン製 RKJXV1224005（相当品）
